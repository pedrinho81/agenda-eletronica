
# Agenda eletrônica


## Rodando localmente

Clone o projeto

```bash
  $ git clone https://github.com/pedrinho81/agenda-eletronica
```

### Client

```bash
  $ cd client
  $ npm install ou yarn install
  $ npm dev ou $ yarn dev
```

### Server

```bash
  $ cd server
  Altere seu servidor de dados MYSQL no server/index.js
  $ npm install ou yarn install
  $ npm dev ou yarn dev
```
